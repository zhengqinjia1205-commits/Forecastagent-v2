# ForecastPro AI需求预测可视化仪表板

基于Streamlit构建的交互式预测可视化工具，提供完整的预测结果展示、模型比较、业务洞察和报告生成功能。

## 🚀 快速开始

### 1. 安装依赖

确保已安装ForecastPro Agent的基础依赖：

```bash
pip install -r requirements.txt
```

安装仪表板额外依赖：

```bash
pip install -r requirements_dashboard.txt
```

或者手动安装核心依赖：

```bash
pip install streamlit>=1.28.0 plotly>=5.17.0 openpyxl>=3.1.0
```

### 2. 运行仪表板

在终端中执行以下命令：

```bash
cd /path/to/forecastpro_agent
streamlit run dashboard.py
```

或者使用Python模块方式：

```bash
python -m streamlit run dashboard.py
```

### 3. 访问仪表板

启动后，浏览器会自动打开仪表板界面（通常为 http://localhost:8501）。如果没有自动打开，请手动访问该地址。

## 📊 功能概述

### 首页
- 系统功能介绍
- 快速开始指南
- 一键使用示例数据运行预测管道

### 数据管理
- **上传数据**: 支持CSV和Excel格式
- **自动识别**: 自动识别时间列和需求列
- **数据预览**: 查看数据结构和统计信息
- **列配置**: 手动指定时间列和需求列

### 预测结果
- **交互式图表**: 展示历史需求、模型拟合值和未来预测
- **多模型对比**: 同时显示多个模型的预测结果
- **置信区间**: 显示未来预测的95%置信区间
- **详细数据**: 查看具体的预测数值

### 模型比较
- **性能排名**: 按MAPE、MAE、RMSE对模型排序
- **可视化对比**: 条形图、雷达图、散点图等多种可视化方式
- **最佳模型**: 高亮显示最佳模型及其性能指标
- **过拟合风险**: 标识模型的过拟合风险等级

### 业务洞察
- **数据概览**: 关键数据统计信息
- **趋势分析**: 需求趋势和季节性模式分析
- **业务建议**: 基于预测结果的行动建议
- **洞察总结**: AI生成的业务洞察点

### 完整报告
- **综合报告**: 包含所有分析结果的完整报告
- **导出功能**: 支持JSON和文本格式报告下载
- **打印友好**: 适合打印和分享的报告格式

### 🤖 AI问答与分析
- **智能问答**: 使用自然语言提问，获取数据洞察和解释
- **预定义模板**: 快速访问常见问题，一键生成回答
- **报告生成**: 根据需求生成特定报告（库存、季节性、模型性能等）
- **时间序列分析问答**: 深入分析季节性、平稳性、自相关等时间序列特征
- **交互式对话**: 保存对话历史，支持连续问答

## 🔧 配置选项

在仪表板侧边栏可以配置以下参数：

1. **数据频率**: 日(D)、周(W)、月(M)、季(Q)、年(Y)
2. **随机种子**: 确保结果可重复性
3. **测试集比例**: 训练集/测试集划分比例（默认20%）

## 📈 使用流程

### 方式一：使用示例数据
1. 在首页点击"使用示例数据运行完整预测管道"按钮
2. 系统自动加载示例数据并运行预测
3. 使用导航栏查看各页面分析结果

### 方式二：使用自定义数据
1. 前往"数据管理"页面
2. 上传CSV或Excel数据文件
3. 配置时间列和需求列
4. 点击"初始化ForecastPro Agent"
5. 前往"预测结果"页面运行预测管道
6. 查看各页面分析结果

### 方式三：使用AI问答功能
1. 加载数据并运行预测管道（方式一或方式二）
2. 前往"🤖 AI问答与分析"页面
3. 输入自然语言问题或点击预定义问题模板
4. 获取AI生成的回答和报告
5. 可下载生成的特定报告

## 🎨 可视化功能

### 支持的图表类型
- **折线图**: 展示时间序列数据和预测
- **条形图**: 比较模型性能指标
- **雷达图**: 多维度模型性能对比
- **散点图**: MAE vs RMSE关系分析
- **面积图**: 置信区间可视化

### 交互功能
- **缩放和平移**: 支持鼠标滚轮缩放和拖拽平移
- **悬停提示**: 鼠标悬停显示详细数据
- **图例开关**: 点击图例显示/隐藏数据系列
- **数据导出**: 支持图表截图和数据导出

## 📱 响应式设计

仪表板支持不同屏幕尺寸：
- **桌面端**: 完整功能，多列布局
- **平板端**: 自适应布局，优化显示
- **手机端**: 单列布局，触摸友好

## ⚠️ 注意事项

1. **数据要求**: 建议至少50个观测值以获得可靠预测
2. **文件格式**: 支持CSV和Excel，确保时间列为标准日期格式
3. **内存使用**: 大数据集可能需要较多内存，建议分批处理
4. **首次运行**: 首次运行可能需要较长时间初始化模型

## 🔍 故障排除

### 常见问题

1. **无法导入ForecastPro Agent**
   ```
   Error: 无法导入ForecastPro Agent
   ```
   解决方案：确保在forecastpro_agent目录下运行，或手动添加路径。

2. **Streamlit未安装**
   ```
   ModuleNotFoundError: No module named 'streamlit'
   ```
   解决方案：执行 `pip install streamlit`

3. **Plotly图表不显示**
   ```
   Error loading Plotly charts
   ```
   解决方案：执行 `pip install plotly`

4. **数据上传失败**
   ```
   Error reading file
   ```
   解决方案：检查文件格式，确保不是损坏文件
